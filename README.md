"Ruleta de seleccion aleatoria de Estudiantes"

¿ Que es este proyecto?
Este proyecto es una aplicacion de consola realizado en c# que permite lo que es seleccionar
estudiantes de forma aleatoria y asiganarles diferente roles, esta orientada a dos eleciones por ejecucion,
utilizando la forma de una ruleta virtual.


Esta pensado para: 
1) Clases
2) Exposiciones
3) Dinamicas grupales
4) Organizacion de roles

¿Que realiza el programa?
1) Realiza registro de estudiantes
2) Asignacion de roles de manera aleatoria
3) Evita repeticiones
4) Muestra resultados en consola
5) Guarda los datos en (txt)
6) Incluye menu interativo
